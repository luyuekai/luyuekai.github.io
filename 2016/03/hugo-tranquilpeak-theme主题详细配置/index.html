hugo-tranquilpeak-theme主题详细配置

三月 27, 2016 发布在 hugo-tranquilpeak-theme, detail setting

四月是你的谎言

hugo-tranquilpeak-theme主题是一个功能强大，界面华丽的hugo主题。
官方描述：A gorgeous responsive theme for Hugo blog framework 废话不多说，直接跨入正题：

先看其默认配置文件：

  • baseurl = “ ”
    • 博客发布时，填写发布的域名；不发布则无所谓。（其实写成什么貌似都没有影响，可能未理解到位）
  • languageCode = “en-us”
  • defaultContentLanguage = “en-us”
    • 页面默认是英文的，如果想变成中文，建议将改为defaultContentLanguage = “zh-cn”,第一个暂时不知道作用，我把改成“zh-cn”，“en-us”都没看出什么变化。
  • title = “Hugo tranquilpeak theme”
    • 默认文章名称，每次自己在新写markdown时，都会覆盖掉的
  • theme = “hugo-tranquilpeak-theme”
    • 指定的主题名称，默认就好。第一个要改了的话，记得把themes里面对应的文件夹名字也换成相同的。否则会报错：找不到相应的文件夹。
  • disqusShortname = “hugo-tranquilpeak-theme”
    • disqus是一个国外的评论系统，和留言类似。这个配置项主要是用来提供留言服务的。具体含义待定。
  • # googleAnalytics = “UA-123-45”
    • Google爸爸的一个服务，默认不启用
  • paginate = 7
    • 一页显示多少条博客，默认7条
  • canonifyurls = true
    • 不明白，保持默认值。

以下挑重点说明：

  • gravatarEmail = “thibaud.lepretre@gmail.com”
    • 国外的一款软件，可以使你的账号使用同一个头像，国内无法使用，因此建议注释掉
  • picture = http://oneumacbh.bkt.clouddn.com/cover1/%E5%B0%91%E5%A5%B3261276,105.jpg
    • 用来设置自己的头像的，gravatarEmail未注释的话，此项会被覆盖。
  • Menu Configuration
    • menu.main
    • menu.links
    • menu.misc 分为三类，不同类在不同的栏子中，根据weight大小来排序，图标可在font库中查找。
  • 支持自定义日期输出格式
  • 支持keywords（没发现在哪显示）
  • sideBar 即为左侧的侧边栏
  • hierarchicalCategories = true
    • 为true时，会在分类哪里看到层级，false则不会
    • bug,分类在我这里用中文会乱码，暂不知道原因
  • coverImage
    • 建议值为自己图片的CDN加载链接，CDN目前采用七牛云较好。
  • thumbnailImage = true
  • thumbnailImagePosition = “bottom”
    • 在index页面显示小照片，设置小照片的位置
  • autoThumbnailImage = true
    • 设置是否自动为小照片
